# MOOHA

오늘의 기분을 이모지로 기록하는 개인 일기장 앱

## About
* ### Project
  * 개인 프로젝트
* ### Duration
  * 1 month

* ### Technical Stack
  * Framework - Flutter
  * DB - Firebase

* ### Functions
  * Google Sign In
  * 각 날짜별 일기 작성
  * 달력에 해당 날짜에 기록된 기분 이모지로 표시
  * 작성한 일기 목록 확인
